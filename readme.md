How to RUN

1. Install de dependencies (requirements.txt).
2. Setup a .env file with POSTGRES settings (port, user, password and database name).
3. Setup in the .env file the AWS credentials.
4. Setup in the .env file the X token for API connection. 
5. Setup in the .env file the KAFKA configuration files.
6. Run Docker compose up -d to start KAFKA, POSTGRES 
7. Run the consumer.py
8. Run the producer.py
